## 什么是钱途？

**钱途 (MoneyPath)** 是一个面向零基础投资者的综合性财商教育平台,通过"知识体系 + 模拟交易 + AI导师"三位一体的方式,帮助用户系统化学习投资理财知识。

**名字由来**: "钱途"谐音"前途",既指财务未来也指人生前景。Path暗示平台提供清晰的学习路径和财务规划导航。

**品牌Slogan**: 你的钱途,明明白白 | Your Money, Your Path

## 核心特色

### 🎮 游戏化学习
闯关式课程设计,20个关卡从零基础到建立个人投资框架,让学习像玩游戏一样有趣

### 📈 无风险实践
虚拟账户 + 真实A股行情数据,在模拟交易中练手,积累经验不亏真金白银

### 🤖 AI私人导师
24/7智能答疑,个性化学习路径推荐,根据你的进度和特点定制成长方案

### 🛠️ 实战工具箱
资产配置计算器、定投模拟器、估值仪表盘等实用工具,理论立即转化为实践

## 学习路径

### 四大学习阶段

#### 阶段一: 财商启蒙 (关卡0.5-5)
建立金钱、储蓄、复利、通胀的基本概念,完成应急金自测

#### 阶段二: 投资入门 (关卡6-10.5)
认识股票、基金、债券、ETF、REITs等投资工具,学习投资成本控制

#### 阶段三: 实战进阶 (关卡11-15)
学会估值、资产配置、风险管理、定投策略,体验危机模拟训练

#### 阶段四: 长期体系 (关卡16-20)
形成个人投资框架、复盘习惯和行为纪律,实现从模拟到真实投资的过渡

## 核心功能

### 📚 知识体系
- 20个精心设计的闯关式课程
- 每关包含互动练习和即时反馈
- 经验值、等级、成就系统激励学习
- 进度追踪和学习数据可视化

### 📈 模拟交易
- 虚拟账户(初始10万元)
- 真实A股/港股/美股行情数据
- 挑战任务系统(如"30天定投")
- 持仓管理和盈亏分析

### 🤖 AI导师
- 智能问答(基于GPT-4o-mini)
- 个性化学习路径推荐
- 投资风险测评
- 学习盲区诊断

### 🛠️ 工具箱
- **资产配置计算器**: 根据风险偏好生成投资组合
- **定投模拟器**: 回测历史定投收益
- **估值仪表盘**: 可视化查看股票/指数估值水平
- **复利计算器**: 演示长期投资的复利效应

## 技术栈

### 前端
- **框架**: Next.js 14 (App Router) + React 18 + TypeScript 5
- **样式**: TailwindCSS 3
- **可视化**: Recharts / ECharts

### 后端
- **框架**: Python 3.11+ + FastAPI 0.110+
- **数据库**: PostgreSQL 15 + Redis 7
- **ORM**: SQLAlchemy 2.0

### AI层
- **模型**: OpenAI GPT-4o-mini
- **对话管理**: LangChain
- **备选**: DeepSeek / 通义千问

### 数据源
- **A股数据**: akshare
- **美股/港股**: Yahoo Finance
