# 内容创作指南

## 1. 写作对象

默认读者是：

- 没有任何理财经验。
- 可能听过股票、基金，但不懂概念。
- 害怕亏钱，容易被“保本高收益”吸引。

## 2. 写作原则

- 先讲生活例子，再引入术语。
- 每节课只讲一个核心概念。
- 不要一次塞太多公式。
- 必须同时讲收益和风险。
- 不能承诺收益，不能荐股。

## 3. 单课结构

```markdown
# 课程标题

## 你会学到什么
一句话说明。

## 生活例子
用工资、存款、买房、购物等场景解释。

## 核心概念
用通俗话解释概念。

## 小实验
一个交互或计算练习。

## 常见误区
列出2-3个。

## 测验
3-5题。

## 一句话总结
用一句话带走知识点。
```

## 4. 示例：复利

错误写法：
> 复利是本金和利息共同参与下一期计息的机制。

推荐写法：
> 小张25岁，刚工作，每月存1000元投资。第一年赚的钱，第二年也会继续帮他赚钱，就像滚雪球一样。到35岁时，他不仅存了12万，还多赚了近3万利息（假设年化8%）。

## 5. 测验题规范

- 每题四个选项。
- 至少一个常见错误选项。
- 答案解析必须解释为什么错。

示例：

```json
{
  "question": "小李月薪8000元，每月存2000元投资，年化收益8%，本金大约几年翻倍？",
  "options": ["3年", "6年", "9年", "20年"],
  "answer": "9年",
  "explanation": "72法则：72 ÷ 8 = 9。不管小李每月存多少，只要年化收益是8%，翻倍时间都是9年左右。"
}
```

## 6. 风险提示规范

涉及以下内容时必须有风险提示：

- 股票收益。
- 基金历史业绩。
- 回测结果。
- AI分析。
- 模拟交易。

推荐措辞：

> 以上内容仅用于学习和模拟，不构成投资建议。真实投资可能亏损本金。

## 7. 术语表规范

每个术语包含：

- 中文名。
- 英文名（如有）。
- 一句话解释。
- 生活化例子。
- 常见误区。
