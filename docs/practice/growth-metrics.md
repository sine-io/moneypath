# 用户成长指标体系

## 1. 核心理念

**不是"学了多少"，而是"会做什么"**

传统学习平台只看完课率、学习时长，但这些指标不能证明用户真的学会了。FinLearn的成长体系聚焦**能力验证**：
- 不是"看完20关"，而是"能独立配置资产"
- 不是"刷了100道题"，而是"模拟账户持续盈利"
- 不是"登录30天"，而是"养成复盘习惯"

## 2. 三维成长模型

## 3. 综合成长指数（FGI: FinLearn Growth Index）

### 3.1 计算公式
```
FGI = 知识维度 × 40% + 实践维度 × 35% + 心智维度 × 25%

满分: 100分
```

**权重说明：**
- 知识是基础（40%）：不懂原理容易亏钱
- 实践是关键（35%）：纸上谈兵没用
- 心智是保障（25%）：再好的策略也需要执行力

### 3.2 等级对应
| FGI分数 | 等级 | 称号 | 画像 |
|---------|------|------|------|
| 0-20 | F | 小白 | 刚开始了解理财 |
| 21-40 | D | 入门者 | 懂基本概念，未实践 |
| 41-60 | C | 实践者 | 开始尝试投资，偶有失误 |
| 61-80 | B | 进阶者 | 有体系，能稳定操作 |
| 81-95 | A | 专家 | 持续盈利，能指导他人 |
| 96-100 | S | 大师 | 形成独特方法论 |
