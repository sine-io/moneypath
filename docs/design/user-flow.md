# 用户流程

## 1. 用户主流程

```text
访问首页
  -> 注册/登录
  -> 风险测评
  -> 新手引导
  -> 第1关课程
  -> 模拟交易开户
  -> AI导师答疑
  -> 工具箱练习
  -> 长期学习/复盘
```

## 2. 核心流程拆解

### 2.1 首次进入

1. 用户打开首页。
2. 看到产品定位和免责声明。
3. 点击“开始学习”。
4. 完成注册。
5. 填写风险测评。
6. 系统推荐第一条学习路径。

### 2.2 学习流程

1. 进入课程地图。
2. 选择当前可解锁关卡。
3. 阅读内容块。
4. 完成交互题。
5. 通过测验。
6. 获得经验值和徽章。
7. 自动解锁下一关。

### 2.3 模拟交易流程

1. 进入模拟账户。
2. 搜索标的。
3. 查看行情和风险提示。
4. 输入买入数量。
5. 确认交易。
6. 查看持仓和交易记录。
7. 后续可卖出或复盘。

### 2.4 AI提问流程

1. 用户进入 AI 导师页。
2. 输入问题或上传截图。
3. 系统附加用户上下文（如持仓、学习进度）。
4. AI 返回解释、建议、相关课程链接。
5. 用户可继续追问。

## 3. 异常流程

### 3.1 资金不足
- 禁止提交订单。
- 提示用户可降低数量或先学习配置课程。

### 3.2 题目未通过
- 允许重试。
- 显示错题解析和对应知识点。

### 3.3 AI触发安全边界
- 不返回荐股或保收益承诺。
- 改为风险教育和一般性解释。

### 3.4 行情源失败
- 使用缓存数据。
- 页面提示数据可能延迟。

## 4. 新手引导策略

- 第一次登录默认只展示最少功能。
- 引导卡片每次只讲一个概念。
- 第一周推荐“复利、通胀、风险收益、资产类别、分散投资”五关。
- 完成一个闭环后再开放更复杂工具。

## 5. 关键交互原则

- 每个页面回答一个问题：我现在该学什么、该练什么、该怎么看。
- 避免一次性展示太多图表。
- 所有金额默认使用人民币。
- 股票/基金代码旁边显示中文名称。
