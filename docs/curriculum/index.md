## 特殊关卡设计

### 🆕 关卡0.5: 应急金检查
- **创新点**: 投资前先建立财务安全网
- **工具**: 应急金自测计算器(月支出×6)
- **阻断机制**: 3个选项引导用户理性决策

### 💥 关卡8.5: 危机模拟
- **创新点**: 体验账户-30%暴跌
- **训练**: 强制情绪日志,建立心理免疫力
- **价值**: 在安全环境预先接种恐慌疫苗

### 💰 关卡10.5: 投资成本全解析
- **创新点**: 量化成本对长期收益的影响
- **震撼案例**: 同样8%收益,30年后费率差导致23万损失
- **实用**: 教你选择低成本工具

### 🚀 关卡18.5: 从模拟到真实
- **创新点**: 解决"学完不敢真投资"的心理障碍
- **指导**: 8个自测问题 + 1000元起步建议
- **过渡**: 48小时检查清单 + 心态对比表

## 学习建议

### 推荐节奏

- **新手**: 每天1关,20天完成基础学习
- **进阶**: 每天2-3关,一周快速通关
- **深度学习**: 配合模拟交易,边学边练

### 学习技巧

1. **不要跳关**: 知识是递进的,跳关容易理解不透
2. **重视互动**: 每关的互动练习比看完文字更重要
3. **真实投入**: 模拟交易要当真做,才能积累经验
4. **坚持复盘**: 每次交易后填写日志,养成思考习惯

### 卡关怎么办?

- **48小时预警**: 系统会主动提供3种学习方式
- **AI导师**: 随时可以提问
- **错题本**: 查看之前的错误,找到薄弱点
