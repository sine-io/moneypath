## 1. 产品概述

### 1.1 产品定位
FinLearn Platform 是一个面向零基础用户的投资理财教育平台，通过游戏化学习、模拟交易和AI辅导三位一体的方式，帮助用户从零到一建立投资理财知识体系。

### 1.2 目标用户

**主要用户画像**

**画像1：职场新人小王**
- 年龄：25岁
- 职业：互联网公司程序员
- 月收入：15,000元
- 痛点：有存款但不知道怎么投资，怕亏钱
- 需求：系统学习投资知识，先模拟练手

**画像2：宝妈张女士**
- 年龄：32岁
- 职业：全职妈妈
- 家庭存款：30万
- 痛点：银行利息太低，不懂股票基金
- 需求：利用碎片时间学习，找到稳健的投资方式

**画像3：大学生小李**
- 年龄：22岁
- 职业：大四学生
- 月生活费：2,000元
- 痛点：想用零花钱投资，但完全没概念
- 需求：从基础概念开始学起

### 1.3 核心价值

- **降低学习门槛**：游戏化闯关，每关5-10分钟
- **零风险实践**：模拟交易，真实数据，不花真钱
- **个性化辅导**：AI导师根据学习进度推荐内容
- **工具化落地**：提供实用的投资计算工具

## 2. 功能需求

### 2.1 知识体系模块

#### 2.1.1 闯关式课程

**功能描述**
用户通过完成关卡学习投资知识，每关包含：
- 3-5分钟知识讲解（图文+短视频）
- 2-3个互动练习
- 1个结业测验（3-5题）

**关卡设计**

**新手村（Level 1-5）**
| 关卡 | 主题 | 学习目标 | 互动方式 |
|------|------|---------|---------|
| 第1关 | 复利的魔力 | 理解72法则 | 复利计算器互动 |
| 第2关 | 通货膨胀 | 理解购买力下降 | 时光机器演示 |
| 第3关 | 风险与收益 | 理解风险对等 | 拖拽式资产配置 |
| 第4关 | 资产类别 | 认识股票/债券/基金 | 卡片翻转游戏 |
| 第5关 | 分散投资 | 理解鸡蛋篮子原理 | 投篮游戏模拟 |

**进阶关卡（Level 6-15）**
- 投资工具深度认知（ETF/指数基金/主动基金）
- 估值基础（PE/PB/ROE）
- K线图入门
- 资产配置实战
- 投资心理学

**验收标准**
- [ ] 用户完成关卡后解锁下一关
- [ ] 测验正确率≥80%才能通过
- [ ] 错误题目显示详细解析
- [ ] 完成关卡获得经验值+成就徽章

#### 2.1.2 知识卡片

**功能描述**
快速查询投资术语和概念，类似"单词卡"。

**内容结构**
```
卡片示例：市盈率（PE）
┌─────────────────────────────┐
│ 市盈率 (PE Ratio)            │
├─────────────────────────────┤
│ 定义：                       │
│ 股价 ÷ 每股收益              │
│                              │
│ 意义：                       │
│ 衡量股票估值高低的指标       │
│                              │
│ 判断标准：                   │
│ • PE < 15: 可能低估          │
│ • PE 15-30: 合理区间         │
│ • PE > 30: 可能高估          │
│                              │
│ 注意事项：                   │
│ 不同行业PE差异大，需横向对比 │
└─────────────────────────────┘
```

**验收标准**
- [ ] 至少包含100个核心术语
- [ ] 支持搜索和分类浏览
- [ ] 支持收藏功能

#### 2.1.3 学习进度追踪

**功能描述**
可视化展示学习进度和成就。

**界面元素**
- 等级系统（LV1 - LV50）
- 经验值进度条
- 成就墙（已解锁/未解锁）
- 学习日历（连续签到天数）
- 知识图谱（已掌握的知识点网络）

**验收标准**
- [ ] 实时更新经验值和等级
- [ ] 完成关卡自动解锁成就
- [ ] 连续学习7天获得奖励徽章

### 2.2 模拟交易模块

#### 2.2.1 虚拟账户系统

**功能描述**
为每个用户创建虚拟投资账户，初始资金10万元。

**核心功能**
- 账户总资产显示
- 可用资金 / 持仓市值
- 累计收益 / 收益率
- 历史交易记录

**验收标准**
- [ ] 注册用户自动获得10万虚拟币
- [ ] 资产数据实时更新
- [ ] 支持重置账户（每月1次）

#### 2.2.2 股票/基金交易

**功能描述**
模拟真实A股交易流程。

**交易规则**
- 真实A股/基金数据（15分钟延迟）
- T+1交易规则
- 手续费模拟（万分之2.5）
- 印花税模拟（千分之1）
- 涨跌停限制（±10%）

**交易界面**
```
┌─────────────────────────────────┐
│ 贵州茅台 (600519)               │
├─────────────────────────────────┤
│ 当前价：1,850.50  涨跌：+2.3%   │
│                                 │
│ 买入数量：[___] 股              │
│ 预计金额：18,505 元             │
│ 手续费：  4.63 元               │
│                                 │
│ [确认买入]  [取消]              │
└─────────────────────────────────┘
```

**验收标准**
- [ ] 支持搜索股票代码/名称
- [ ] 显示实时行情（延迟15分钟）
- [ ] 交易成功后更新持仓
- [ ] 拒绝无效交易（资金不足等）

#### 2.2.3 持仓管理

**功能描述**
查看和管理持仓股票/基金。

**持仓列表**
| 代码 | 名称 | 持仓量 | 成本价 | 现价 | 盈亏 | 盈亏率 |
|------|------|--------|--------|------|------|--------|
| 600519 | 贵州茅台 | 100股 | 1,800 | 1,850 | +5,000 | +2.78% |
| 510300 | 沪深300ETF | 1000份 | 4.50 | 4.65 | +150 | +3.33% |

**操作功能**
- 点击持仓进入详情页
- 支持卖出操作
- 显示历史买入记录

**验收标准**
- [ ] 实时计算盈亏
- [ ] 支持按盈亏率排序
- [ ] 显示总持仓市值

#### 2.2.4 挑战任务

**功能描述**
设定投资挑战目标，完成后获得奖励。

**任务示例**
| 任务名称 | 目标 | 难度 | 奖励 |
|---------|------|------|------|
| 初试牛刀 | 用10万赚到12万 | ⭐ | 500 XP |
| 稳健配置 | 构建60/40组合 | ⭐⭐ | 1000 XP |
| 熊市生存 | 回撤控制在-10%内 | ⭐⭐⭐ | 2000 XP |
| 跑赢大盘 | 收益超过沪深300 | ⭐⭐⭐⭐ | 5000 XP |

**验收标准**
- [ ] 任务进度实时追踪
- [ ] 完成任务自动发放奖励
- [ ] 支持每日/每周/每月任务

### 2.3 AI导师模块

#### 2.3.1 智能问答

**功能描述**
用户可以向AI导师提问，获得个性化解答。

**对话示例**
```
用户: "为什么我的基金一直亏？"

AI: "让我帮你分析一下 🔍
   
   看了你的持仓，发现3个问题：
   
   ❌ 问题1：过度集中
   你的基金80%都是医疗主题，行业过于单一
   
   ❌ 问题2：买入时机
   你在今年2月高点买入，正好是医疗板块顶部
   
   ❌ 问题3：没有分批
   10万一次性买入，没有分散风险
   
   💡 建议方案：
   1. 卖出50%医疗基金，换成沪深300ETF
   2. 剩余资金分6个月定投
   3. 每月第一个交易日投入1.5万
   
   [查看详细方案] [模拟回测]"
```

**功能要求**
- 支持文字对话
- 支持上传持仓截图分析
- 支持多轮对话
- 支持历史对话查询

**验收标准**
- [ ] 回复延迟<3秒
- [ ] 支持上下文理解
- [ ] 提供可操作的建议
- [ ] 敏感话题自动拒绝（如荐股）

#### 2.3.2 风险测评

**功能描述**
通过问卷评估用户风险承受能力。

**问卷设计（10题）**
1. 您的年龄？
2. 您的投资经验？
3. 如果投资亏损20%，您会？
4. 您的投资目标是？
5. 您能承受的最大亏损？
...

**评估结果**
- 保守型（建议配置：20%股票 + 80%债券）
- 稳健型（建议配置：40%股票 + 60%债券）
- 平衡型（建议配置：60%股票 + 40%债券）
- 积极型（建议配置：80%股票 + 20%债券）
- 激进型（建议配置：100%股票）

**验收标准**
- [ ] 完成测评生成报告
- [ ] 报告包含资产配置建议
- [ ] 支持每3个月重新测评

#### 2.3.3 个性化推荐

**功能描述**
基于用户行为推荐学习内容。

**推荐逻辑**
- 完成第3关 → 推荐"资产配置实战"关卡
- 多次搜索"PE" → 推荐"估值基础"课程
- 模拟交易亏损>10% → 推荐"风险管理"专题

**验收标准**
- [ ] 每日推荐3条内容
- [ ] 推荐内容与用户行为相关
- [ ] 支持"不感兴趣"反馈

## 3. 非功能需求

### 3.1 性能要求
- 页面加载时间 < 2秒
- AI回复延迟 < 3秒
- 支持1000并发用户
- 数据更新延迟 < 15分钟（实时行情）

### 3.2 安全要求
- HTTPS加密传输
- 密码bcrypt加密存储
- SQL注入防护
- XSS攻击防护
- CSRF防护

### 3.3 可用性要求
- 系统可用性 ≥ 99.5%
- 数据备份：每日自动备份
- 灾难恢复时间 < 4小时

### 3.4 兼容性要求
- 浏览器：Chrome 90+, Firefox 88+, Safari 14+
- 移动端：响应式设计，兼容iOS/Android
- 屏幕分辨率：≥1366x768

## 6. 附录

### 6.1 竞品分析
| 产品 | 优势 | 劣势 |
|------|------|------|
| 蚂蚁财富 | 用户量大，资源多 | 重导流，教育弱 |
| 雪球 | 社区活跃 | 门槛高，小白劝退 |
| 且慢 | 投资理念好 | 付费门槛高 |

### 6.2 术语表
- **PE**：市盈率，股价÷每股收益
- **PB**：市净率，股价÷每股净资产
- **ROE**：净资产收益率，净利润÷净资产
- **ETF**：交易型开放式指数基金
- **定投**：定期定额投资
