**场景：** 每关学习过程中
**目标：** 巩固刚学的概念，立即验证理解

### 1.2 关卡测验（阶段性验证）
**场景：** 每关结束时
**目标：** 验证学习效果，≥80分才能过关

#### 题型设计
**单选题（60%）**
```
Q1: 小明有10万元，年化收益8%，根据72法则，
    多少年后本金能翻倍？

A. 7年
B. 9年 ✓
C. 12年
D. 15年

👉 你的答案：B
✅ 回答正确！72 ÷ 8 = 9年

💡 知识点：72法则可快速估算翻倍时间
```

**场景判断题（30%）**
```
Q2: 小王每月定投1000元买基金，遇到股市下跌，
    他应该怎么做？

A. 立即停止定投，等涨回来再买
B. 继续定投，因为这时可以买到更多份额 ✓
C. 全部卖出，避免更大损失

👉 你的答案：B
✅ 回答正确！定投的核心是分批买入，下跌时
   更低的价格反而能降低平均成本。

💡 延伸阅读：[第14关：定投策略]
```

**计算题（10%）**
```
Q3: 如果通货膨胀率是3%，100元存银行1年
    (年利率1.5%)，实际购买力变成多少？

输入答案: [98.5] 元

✅ 正确！
📝 计算过程：
   名义收益 = 100 × 1.5% = 1.5元
   通胀损失 = 100 × 3% = 3元
   实际购买力 = 100 + 1.5 - 3 = 98.5元
```

#### 反馈机制
**即时反馈：**
- 每题提交后立即显示对错
- 错误时展开详细解析+关联知识点链接
- 显示全站用户该题正确率（例如："73%的用户答对了这题"）

**重做机制：**
- 首次<80分：允许重做，但只能重做错题
- 连续3次不过：提示"要不先回顾一下课程内容？"并提供快速跳转
- 最多重做5次后强制通过，但标记"需复习"

### 1.3 综合实战（模拟交易）
**场景：** 完成第5关后开放
**目标：** 在真实市场环境下应用知识

#### 实战任务体系
| 阶段 | 任务 | 目标 | 验收标准 |
|------|------|------|---------|
| 启蒙期 | 我的第一笔投资 | 完成1笔基金买入 | 持仓>0 |
| 入门期 | 资产配置练习 | 持有≥3种不同类型资产 | 股票+基金+债券 |
| 进阶期 | 30天定投挑战 | 坚持定投30天 | 交易记录≥4笔 |
| 体系期 | 组合优化 | 调整持仓使夏普比率>1.5 | 系统自动计算 |

## 6. 核心指标

| 指标 | 目标值 | 监控频率 |
|------|--------|---------|
| 测验平均通过率 | 75-85% | 每天 |
| 交易日志填写率 | >90% | 每周 |
| 错题本使用率 | >40% | 每周 |
| 周复盘查看率 | >50% | 每周 |
| 实战任务完成率 | >60% | 每月 |

**预警机制：**
- 测验通过率<70%：题目太难，需调整
- 交易日志填写率<80%：流程太繁琐，需简化
- 错题本使用率<30%：入口不明显或功能不实用
