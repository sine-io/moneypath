# 增长策略

## 1. 冷启动目标

前3个月目标不是大规模增长，而是验证：

- 0基础用户是否看得懂。
- 课程是否能被持续完成。
- 模拟交易是否增强粘性。
- AI导师是否有真实价值。

## 2. 第一批用户来源

| 渠道 | 方式 |
|---|---|
| 朋友圈/微信群 | 找10-30个真实小白内测 |
| 小红书 | 发“理财小白第1课”内容 |
| B站 | 做短视频：复利、通胀、ETF |
| 知乎 | 回答“理财入门”问题 |
| Telegram/社群 | 邀请朋友试用反馈 |

## 3. 内容营销主题

- “0基础理财先学这5个概念”。
- “为什么保本高收益都是坑”。
- “用虚拟10万元练习买ETF”。
- “普通人怎么做资产配置”。
- “定投到底赚钱吗？用历史数据模拟”。

## 4. 留存机制

- 每日一题。
- 连续学习徽章。
- 每周模拟组合复盘。
- AI导师生成学习提醒。
- 完成关卡解锁工具。

## 5. 北极星指标

建议 MVP 阶段关注：

- 完成第1课用户占比。
- 完成第5课用户占比。
- 第7日留存。
- 模拟交易首单转化率。
- AI提问次数/用户。

## 6. 不建议早期做

- 大规模投广告。
- 复杂裂变活动。
- 承诺收益吸引用户。
- 用荐股作为卖点。
