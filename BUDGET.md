# 成本预算

## 1. 开发成本

### 人力
如果由 1 人兼职开发：
- 预计周期：6 个月
- 主要成本：时间成本
- 若按外包估价：通常远高于自研，不建议一开始外包全部

### 工具与服务
| 项目 | 方案 | 月成本 |
|---|---|---:|
| 服务器 | 2核4G 云服务器 | 100 元 |
| 数据库 | 自建 PostgreSQL | 0 元 |
| 缓存 | 自建 Redis | 0 元 |
| 域名/CDN | 基础配置 | 50 元 |
| AI 调用 | 低价模型优先 | 200-800 元 |
| 监控 | 开源方案 | 0-50 元 |

## 2. MVP 月成本

| 场景 | 月成本 |
|---|---:|
| 极简版 | 150 元 |
| 标准版 | 350-650 元 |
| AI 高频使用版 | 800-1500 元 |

## 3. 年成本估算

按标准版估算：
- 650 元/月 × 12 = 7800 元/年

## 4. 成本控制建议

- AI 默认使用低成本模型。
- 高频问答使用缓存。
- 行情数据尽量走免费源。
- 先做 Web，不急着做 App。
- 先做内容和核心功能，不先做社区。

## 5. 商业化前提

如果后续要收费，建议优先考虑：

1. 高级课程订阅。
2. 进阶回测工具。
3. 组合复盘报告。
4. 企业/培训版本。

## 6. 不建议早期投入

- 原生 App 大而全。
- 社区广场和复杂私信系统。
- 实盘交易接入。
- 复杂的风控和监管接口。
