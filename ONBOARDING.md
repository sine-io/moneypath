### 1.2 快速测评（90秒）
**目的：** 了解用户起点，提供个性化路径

**问题设计（3题）：**
1. 朋友跟你说他买基金赚了30%，你的第一反应是？
   - [ ] 基金是什么？能吃吗？（→ 从关卡0.5开始）
   - [ ] 我也想买！但不知道怎么操作（→ 从第2关开始）
   - [ ] 先等等，我得研究下风险（→ 从第6关开始）

2. 如果有1万元闲钱放着半年不用，你会？
   - [ ] 存银行活期，安全第一
   - [ ] 不知道怎么办，但想让钱增值
   - [ ] 买货币基金或短期理财

3. 你最担心的是？
   - [ ] 完全不懂，怕被骗或做错
   - [ ] 怕亏钱，想学稳健的方法
   - [ ] 想提高收益，但不知道怎么分析

**输出：** 生成专属学习路径

## 2. 阶段目标与验收标准

### 阶段一：财商启蒙（第1-5关）
**学习目标：**
- 理解复利、通胀、风险收益的基本概念
- 建立"储蓄不等于投资"的认知
- 完成第一笔模拟交易（第1关后即可体验）

**验收标准：**
- [ ] 完成关卡0.5"应急金检查",通过应急金自测并做出选择
- [ ] 完成5关，每关测验≥80分
- [ ] 累计学习时长≥30分钟
- [ ] 已完成第一笔投资体验（第1关后自动引导）
- [ ] 能用自己的话解释"复利"和"通胀"

**通关奖励：**
- "财商启蒙"认证徽章
- 解锁模拟交易高级功能
- 开放进阶课程（第6-10关）
- **目标进度更新：** "你已掌握基础财务知识，目标可行性：30% → 50%"

### 阶段二：投资入门（第6-10关）
**学习目标：**
- 认识股票、基金、ETF、债券、REITs
- 理解不同资产的风险收益特征
- 能独立分析一只基金

**验收标准：**
- [ ] 完成5关，平均分≥85分
- [ ] 持有至少3种不同类型资产
- [ ] 完成1次"资产配置"作业（提交持仓截图+配置思路）
- [ ] 通过"投资工具识别"闯关游戏（90%准确率）

**通关奖励：**
- "投资学徒"认证
- 解锁估值工具包
- 开放真实市场数据查询
- **目标进度更新：** 
  ```
  "完成投资入门！你现在能识别和分析不同投资工具了"
  
  目标：10年后子女教育金50万
  
  按当前掌握的知识：
  - 定投沪深300指数基金
  - 每月投资3500元
  - 预期年化8%
  
  → 10年后可达到：64万元
  ✅ 超出目标28%，还有14万元富余！
  
  目标可行性：50% → 70%
  ```

### 阶段三：实战进阶（第11-15关）
**学习目标：**
- 学会看PE/PB/ROE等估值指标
- 掌握资产配置方法
- 建立定投纪律

**验收标准：**
- [ ] 完成5关，平均分≥85分
- [ ] 提交1份个人资产配置方案（含股债比例、风险分析）
- [ ] 坚持定投模拟≥30天
- [ ] 完成1次完整的投资复盘（买入理由→过程记录→卖出决策）

**通关奖励：**
- "投资实践者"认证
- 解锁组合分析工具
- 获得AI导师"深度问诊"10次
- **目标进度更新：**
  ```

  每月投资2000元，30年后预计：
  → 280万元（超出目标40%）
  
  按当前掌握的知识：
  "你的退休计划可行性从45%提升到85%"
  "继续学习长期投资策略，可再提升至95%"
  ```
